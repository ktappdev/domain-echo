# Changelog

All notable changes to Domain Echo will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-09

### Added
- Initial release
- Anonymous real-time chat for every website domain
- Auto-generated pop culture usernames
- Real-time message synchronization using Convex
- Clean, modern UI with dark theme
- Privacy-first design (no tracking, no data collection)
- Chrome Manifest V3 support

### Features
- Domain-based chat rooms
- Anonymous usernames (2.4M+ combinations)
- Real-time WebSocket connections
- Message persistence
- Responsive popup interface
- Active user count display

### Technical
- Convex backend integration
- ConvexClient for real-time subscriptions
- Local Convex library bundling (CSP compliant)
- Service worker background script
- Chrome storage API for username persistence

## Version Numbering Guide

- **MAJOR** version (X.0.0): Incompatible API changes or major redesigns
- **MINOR** version (0.X.0): New features, backwards compatible
- **PATCH** version (0.0.X): Bug fixes, backwards compatible

Examples:
- 1.0.0 → 1.0.1: Fixed username generation bug
- 1.0.1 → 1.1.0: Added emoji support
- 1.1.0 → 2.0.0: Complete UI redesign
