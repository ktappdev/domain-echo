HOW IT WORKS IN 30 SECONDS
==========================

1. User clicks extension icon
2. Extension detects current domain
3. Popup shows chat room for that domain
4. User types message and sends
5. Message stored in Supabase
6. Supabase broadcasts to all in that room
7. Message appears in real-time for everyone
8. Other users can reply
9. User switches domains = new chat room
10. Previous room forgotten locally, messages remain in database
